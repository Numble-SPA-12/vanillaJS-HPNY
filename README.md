## vanilla JavaScript 만을 사용해 신년 메세지를 주고받는 SPA 프로젝트 (numble challenge)

[배포 링크](http://happy-new-year-project.s3-website.ap-northeast-2.amazonaws.com/)

## 실행 방법

```bash
yarn install

yarn dev
```

## 사용 기술
- JavaScript
- scss
- webpack (번들러)
- axios

## 얻은 경험
- webpack 처음부터 설정
- Vanilla JavaScript로 Router부터 구현하여 SPA 프로젝트 완성
- XSS 공격 방어
- AWS S3로 정적 페이지 배포
- gighub action을 통해 CI/CD 적용
